# Bot Automatizacion Camaras

El repositorio *bot-automatizacion-camaras* está orientado a automatizar la apertura y manipulación de cámaras en  "Tienda Esperanza". El objetivo principal es facilitar tareas repetitivas relacionadas con la gestión de cámaras, usando automatización de interfaz gráfica (GUI) con la librería pyautogui.

## Que hace
Automatiza pasos para abrir, mover y colocar cámaras en posiciones específicas dentro de la tienda, ejecutando scripts por pasos (step-1.py, step-2.py, etc.).

- Cada paso implementa una clase que hereda de una interfaz común (Interface_step), y realiza acciones como:
- Hacer clic en las cámaras (Step1)
- Abrir el menú de cámaras (Step2)
- Mover cámaras a lugares predeterminados, simulando el movimiento del mouse y clics (Step3)
- Acciones adicionales de manipulación visual (Step4,Step5,Step6,etc) se pueden implementar


El flujo consiste en detectar imágenes en pantalla (por ejemplo, botones o áreas de la aplicación de cámaras) y simular movimientos y clics del mouse, de manera secuencial y automatizada.
